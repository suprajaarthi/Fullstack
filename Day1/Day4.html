Welcome to HTML Exercise

Create a simple HTML page with the following elements:

  1. Create a section with the id "intro" and a heading with the text "Introduction".
  2. Inside the "intro" section, create a paragraph with the text "This is a sample HTML exercise."
  3. Create a div with the class "container". Inside the div, create a form with the following elements:
    • An input field of type "text" with the class "form-control" and placeholder "Enter your name".
    • A button with the class "btn" and text "Submit".
  4. Create a section with the id "content" and a heading with the text "Content".
  5. Inside the "content" section, create a paragraph with the text "Lorem ipsum dolor sit amet, consectetur adipiscing elit."
  6. Add a CSS rule to change the background color of the body to #f2f2f2 and the font color to #333.
  7. Add a CSS rule to change the color of the second-level heading (h2) to #007bff.
  8. Add a CSS rule to style the "container" class with a white background, padding of 20px, margin-bottom of 20px, and a border radius of 5px.
  9. Add a CSS rule to style the "form-control" class with a margin-bottom of 10px.
  10. Add a CSS rule to style the "btn" class with a background color of #007bff, text color of #fff, padding of 10px 20px, border radius of 5px, and a cursor pointer.
