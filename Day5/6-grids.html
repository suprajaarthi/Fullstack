Grid Elements

An HTML element becomes a grid container when its display property is set to grid or inline-grid. The vertical lines of grid items are called columns. The horizontal lines of grid items are called rows. You can adjust the gap size by using one of the following properties: column-gap row-gap gap

A Grid Layout must have a parent element with the display property set to grid or inline-grid.

Direct child element(s) of the grid container automatically becomes grid items.
