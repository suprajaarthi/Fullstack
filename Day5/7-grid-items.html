A Five Items Grid Layout

By default, a container has one grid item for each column, in each row, but you can style the grid items so that they will span multiple columns and/or rows. The grid-column property defines on which column(s) to place an item. You define where the item will start, and where the item will end. The grid-row property defines on which row to place an item. You define where the item will start, and where the item will end.
